# 数学物理笔记（更新中）

本项目利用 AI Coding 技术，从零开始，渐进式构建一份严谨的、成体系的“中文”数学物理知识库，重要知识点会被形式化验证，最终生成一部分完整的 PDF 文档。虽然内容不够深入，但侧重于知识点之间的联系，构建起一个大体的框架，以便日后深入学习补充细节。

**需要强调，笔者是一位软件开发者，未受过专业系统的数学训练，其中的内容难免有疏漏与错误，欢迎批评指出，笔者会即时更正**。
