# 数理逻辑

数理逻辑学源自 19 世纪末 20 世纪初的“第三次数学危机”，它要解决的是数学严谨性的问题。

**数理逻辑 Mathematical Logic**：数理逻辑用人工语言形式化经典逻辑中的演绎逻辑，将经典演绎逻辑中的逻辑规律翻译为形式系统的规律，然后研究这个形式系统的数学性质。

经典逻辑学和数理逻辑学做个简单对比：

|     | 经典逻辑学 | 数理逻辑学 |
| --- | --- | --- |
| 研究内容 | 思维的形式 | 形式系统的数学性质 |
| 对象语言 | 自然语言 | 人工语言 |
| 元语言 | 自然语言 | 自然语言 |

数理逻辑的研究核心分为四个部分：
1. 证明论 Proof Theory：研究形式证明
2. 模型论 Model Theory：研究形式语言与其数学结构（模型）之间的关系
3. 递归论 Recursion Theory：研究可计算性和可递归函数
4. 集合论 Set Theory：研究集合及其性质
