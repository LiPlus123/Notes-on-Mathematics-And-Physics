# 数理逻辑

**数理逻辑 Mathematical Logic**：数理逻辑是用人工语言形式化演绎推理，并用数学方法研究形式语言、形式证明、形式语义、可计算性和集合基础的学科。它把经典逻辑中的推理规律翻译为形式系统中的符号规则，再研究这些形式系统本身的数学性质。

数理逻辑不是把自然语言中的一切思维活动都形式化，而是首先选择适合数学推理的部分：命题、谓词、量词、公理、推理规则和证明。经过形式化后，“一个推理是否有效”不再只依赖直观理解，而可以转化为“某个公式是否可由规则推出”“某个公式是否在所有模型中为真”等数学问题。

## 数理逻辑的诞生

19 世纪末到 20 世纪初，数学基础中出现了一系列困难：无穷集合的使用需要更严格的说明，分析学需要严密的实数基础，集合论悖论暴露了朴素集合概念的风险。这些问题通常被称为数学基础危机。数理逻辑就是在这种背景下发展起来的，它要解决的核心问题是：数学证明为什么可靠？数学对象和数学命题应当建立在怎样的语言、公理和推理规则之上？

在这场基础研究中，出现了不同的哲学立场和技术路线，其中直觉主义和形式主义对数理逻辑的发展影响尤其大。

**直觉主义 Intuitionism**：直觉主义认为数学对象必须由构造活动给出，数学命题的真应当以能够给出证明或构造为依据。对直觉主义而言，证明“存在某个对象”通常需要实际构造出这个对象；在没有构造性证明时，不能简单接受“一个命题或其否定必有一个为真”的排中律。

**形式主义 Formalism**：形式主义认为数学可以被看作符号及其变形规则组成的形式系统。数学证明是从公理出发，按照明确推理规则得到公式的有限符号过程；数学基础研究的重要任务是说明这些形式系统的一致性、可靠性和适用范围。

直觉主义和形式主义的区别可以概括为：直觉主义重视数学对象和证明的构造意义，形式主义重视公理、符号和推理规则的形式结构。二者对经典逻辑的态度也不同：直觉主义限制排中律和非构造性存在证明的使用；形式主义通常允许在给定形式系统内使用经典推理，但要求研究该系统是否会推出矛盾。

## 研究对象

数理逻辑的研究对象不是某个具体数学分支中的对象，而是数学语言、数学证明和数学理论本身。它主要研究以下内容：

1. **形式语言 Formal Language**：研究符号、项、公式和语法规则，说明哪些表达式是合法的数学表达式。
2. **形式系统 Formal System**：研究公理、推理规则和形式证明，说明哪些命题可以在系统中被证明。
3. **形式语义 Formal Semantics**：研究语言与数学结构之间的解释关系，说明公式在模型中何时为真。
4. **可计算性 Computability**：研究哪些函数、问题或推理过程可以由机械程序完成。
5. **集合基础 Set-theoretic Foundation**：研究集合及其公理系统，为数学对象提供统一的基础语言。

数理逻辑的研究核心通常分为四个部分：

1. **证明论 Proof Theory**：研究形式证明、证明系统以及证明的结构性质。
2. **模型论 Model Theory**：研究形式语言与其数学结构（模型）之间的关系。
3. **递归论 Recursion Theory**：也称可计算性理论，研究可计算性、可判定性和可递归函数。
4. **集合论 Set Theory**：研究集合、公理系统以及数学对象的集合论基础。

## 与经典逻辑的区别

经典逻辑学和数理逻辑学可以作如下对比：

|     | 经典逻辑学 | 数理逻辑学 |
| --- | --- | --- |
| 研究目标 | 分析思维和推理的形式规律 | 形式化推理，并研究形式系统的数学性质 |
| 研究内容 | 词项、命题、推理及其有效性 | 形式语言、形式证明、形式语义、可计算性、集合基础 |
| 对象语言 | 自然语言 | 人工语言 |
| 元语言 | 自然语言 | 自然语言 |
| 方法特点 | 主要依赖概念分析和推理形式分析 | 使用符号化、公理化和数学证明 |
| 典型问题 | 一个自然语言推理是否有效 | 一个公式是否可证、是否语义有效、一个理论是否一致或完全 |

二者的关系是继承和转化关系：数理逻辑继承了经典逻辑对有效推理的关心，但它不再直接研究自然语言中的推理表达，而是先构造严格的形式语言和形式系统，再用数学方法研究这些系统的性质。因此，经典逻辑更接近“关于思维形式的逻辑学”，数理逻辑更接近“关于形式系统的数学”。
